# E-Commerce-Back-End

## User Story

```md
AS A manager at an internet retail company
I WANT a back end for my e-commerce website that uses the latest technologies
SO THAT my company can compete with other e-commerce companies
```

## Description
The purpose of this repository is to create the backend of an e-commerce site. This application uses Express, Sequelize, and node to connect with a MySQL database. With this application, the user is able to use the CRUD method with the REST APIs. Below are links to the GitHub Repository and the walkthrough video for this application.
